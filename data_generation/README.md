# Заметки по генерации данных для базы данных



**Установка и настройка модуля Faker:**
`pip install Faker`

✔ Структура файлов:
1. `generate_users.py` — для генерации данных пользователей.
2. `generate_students.py` — для генерации данных студентов.
3. `generate_directions.py` — для генерации направлений.
4. `generate_subjects.py` — для генерации предметов.
5. `generate_olympiads.py` — для генерации данных олимпиад.
6. `generate_olympiad_registrations.py` — для генерации регистраций на олимпиады.
